# Case Study Video Presentation

* Selected Topic: AnyDesk Breach

## Damages

* Hacker accessed company's production systems.
* Stole source code and private code signing keys/certificates.
* A service disruption, maintenance required.
* User password exposure.

## Responses

* Activate a response plan with CrowdStrike, an cybersecurity firm.
* Revoked security-related certificates.
* Re-mediated and replaced the system as necessary.
* Ask users to update to the latest.
* Revoke all passwords on web portal and suggests a change.
